Search for a word from the dictionary using the button and writing its definition in the textBox using OleDb and WinForms in C++/CLI

1. Create a WinForms application in Visual Studio and add a button, a textBox and an OleDbConnection.

2. Add the following code in the button’s click event handler.

// Create a database connection
OleDbConnection^ con = gcnew OleDbConnection("Provider=Microsoft.ACE.OLEDB.12.0;Data Source=Dictionary.accdb");

// Open the connection
con->Open();

// Retrieve the word from the textBox
String^ searchWord = textBox->Text;

// Create a SQL query to search for the word
String^ query = "SELECT definition FROM dictionary WHERE word = '" + searchWord + "'";

// Create an OleDbCommand object
OleDbCommand^ cmd = gcnew OleDbCommand(query, con);

// Execute the query
OleDbDataReader^ reader = cmd->ExecuteReader();

// Check if the word was found in the database
if (reader->Read()) 
{
    // Display the definition of the word
    MessageBox::Show(reader["definition"]->ToString());
}
else
{
    // Display an error message
    MessageBox::Show("The word was not found in the dictionary!");
}

// Close the connection
con->Close();

3. Run the application and enter a word in the textBox. When you click the button, the definition of the word will be displayed in a message box.
